Add a reblip button to your posts.

== Description ==

This plugin adds a reblip button to your posts so that your readers can easily blip about your posts and you can see in one place what they blip about. 

For backwards compatibility, if this section is missing, the full length of the short description will be used, and
Markdown parsed.

== Installation ==

1. reblip plugin can be downloaded directly from your wordpress installation

== Frequently Asked Questions ==

= How can I change the position of the reblip button? =

The div with the button has padding inlined but you can implement css rules for that div. It's id is reblip.
